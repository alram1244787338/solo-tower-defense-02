# 🏰 Tower Defense

一个用纯 Canvas + JavaScript 实现的塔防小游戏，无框架无依赖。

## 玩法

- 点击右侧塔按钮或按 `1/2/3` 选择塔种，再点地图空地放置
- 点击已放置的塔查看信息，可升级或出售
- 按 `空格` 开始下一波（倒计时中可跳过）
- 怪物走到终点扣命，命没了游戏结束
- 打完所有波次胜利

### 快捷键

| 按键 | 功能 |
|------|------|
| `1` / `2` / `3` | 选择箭塔 / 炮塔 / 冰塔 |
| `空格` | 开始下一波 / 跳过倒计时 |
| `P` | 暂停 / 继续 |
| `M` | 静音 / 取消静音 |
| `Esc` / 右键 | 取消选择 |

## 三种塔

| 塔种 | 特点 | 造价 | 伤害 | 射程 |
|------|------|------|------|------|
| 🏹 箭塔 | 攻速快，单体伤害低 | 50 | 8 | 140 |
| 💣 炮塔 | 攻速慢，范围溅射伤害 | 120 | 30 | 110 |
| ❄️ 冰塔 | 减速敌人，伤害低 | 90 | 4 | 120 |

塔可升级至 3 级，每级伤害×1.6、射程×1.15、攻速×0.85、费用×1.4。
出售返还累计投入的 60% 金币。

## 波次

- 共 10 波，难度递增，每波结束奖励 30 金币
- 波次间有 5 秒倒计时，可点击跳过
- 四种敌人：小兵、疾行者、坦克、BOSS
- 生命归零则失败，全部通关则胜利

## 运行

直接用浏览器打开 `index.html` 即可。无需安装任何依赖。

也可以起个本地服务器：
```bash
python -m http.server 8080
# 然后访问 http://localhost:8080
```

## 技术

- **渲染**：HTML5 Canvas 2D
- **音效**：Web Audio API 实时合成（开火、命中、漏怪、胜负）
- **零依赖**：纯原生 JavaScript，无框架无 npm 包
- **模块分离**：按职责拆分成多个文件，便于维护

## 文件结构

```
js/
├── config.js   # 游戏配置（塔、怪物、波次、路径）
├── audio.js    # Web Audio API 音效
├── map.js      # 地图路径、位置计算
├── enemy.js    # 敌人类（移动、血量、减速）
├── bullet.js   # 子弹和爆炸效果
├── tower.js    # 塔类（攻击、升级、出售）
├── wave.js     # 波次管理（生成、倒计时）
├── ui.js       # DOM UI 交互
└── game.js     # 主循环，整合所有模块

test.html       # 单元测试（46 个用例，浏览器直接跑）
index.html      # 游戏主页面
```
